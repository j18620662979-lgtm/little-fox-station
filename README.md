# 沈执の小狐狸打卡站（Vue 3 + Vite + Tailwind + Pinia）

## 运行方式
1) 安装依赖
```bash
npm i
```

2) 本地启动
```bash
npm run dev
```

3) 打包
```bash
npm run build
npm run preview
```

## 已实现的关键点（对应你的文档）
- SPA 单页应用，移动端自适应
- 顶部：承诺栏 + 总打卡数据 + 小狐狸头像 + 音乐按钮（占位）
- 目标卡片：最多显示 3 个；勾选后可多目标一起打卡；修改仅限一次；删除二次确认；归档
- 打卡逻辑：每次打卡总数 +1；目标累计 +1；本地持久化 localStorage
- 日历：打卡日期标记；点日期可查看当天完成目标（toast）
- 左侧导航（PC 常驻 / 移动端抽屉式）：信件/礼物/成就/归档/统计
- 信件/礼物：满足打卡总数后可手动解锁；解锁消耗打卡数；解锁后可无限查看；不可重复解锁
- 关键操作均带确认弹窗

## 你可以接下来想改的几个小点
- 把 🦊 小狐狸像素图改成真正图片：放到 `src/assets/` 再在 TopBar 里用 `<img />`
- 音乐按钮：接入 `<audio>` + playlist
- 日历详情：做成弹窗（目前用 toast 显示）
- 统计图：换成 ECharts/Chart.js（现在是轻量版）
